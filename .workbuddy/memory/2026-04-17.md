## Side Panel UI 重构（03:49）

**用户需求**：把启动服务和 DeepSeek Agent 移到设置里，右上角放齿轮设置按钮，中间放 DeepSeek 输出和 Queue Server 执行结果日志，底部放聊天输入框。

**修改文件**：
- `sidepanel.html`：完全重写，双视图结构（主视图=头部+日志区+输入框，设置视图=服务启停+进化面板），深色科技风
- `sidepanel.js`：完全重写，聊天发送（通过 background 中转）、日志渲染（分角色气泡：用户/ai/系统/任务）、视图切换、Native Host 通信、Markdown 渲染、打字指示器
- `background.js`：新增 `sidepanel_chat` 消息路由和 `handleSidepanelChat` 函数

**关键设计决策**：
- 单页面 CSS 视图切换，而非多 HTML 页面，避免状态同步问题
- 聊天消息走 background.js 中转（`sidepanel_chat` → `handleSidepanelChat` → content script），复用已有的 DeepSeek 标签页查找/创建逻辑
- 日志消息类型：用户消息(右对齐蓝紫渐变)、AI回复(左对齐深色卡片)、系统消息(居中虚线框)、任务结果(左对齐绿色/红色边框)
- 自动滚动：用户向上滚动时暂停，滚到底部时恢复
